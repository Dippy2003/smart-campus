📌 Smart Campus Resource Management System
👨‍💻 Module: Facilities & Assets Catalogue

This project is part of the Smart Campus System developed using:

Spring Boot (Backend)

React + Vite (Frontend)

MySQL Database

This module manages campus resources such as:

Lecture Halls

Labs

Meeting Rooms

Equipment

Backend (Spring Boot)

✔ Create Resource
✔ View All Resources
✔ View Resource by ID
✔ Update Resource
✔ Delete Resource
✔ Filter by Type
✔ Filter by Status
✔ Filter by Location
✔ Search Resources

Frontend (React)

✔ Resource List Page
✔ Search & Filter
✔ Resource Details Page
✔ Admin Resource Management
✔ Add Resource
✔ Edit Resource
✔ Delete Resource
